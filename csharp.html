Başlangıç Seviyesi Konular

Bu bölümde, C# programlama dilinin temellerini öğreneceksiniz.

1. C#'a Giriş

C#, güçlü ve modern bir programlama dilidir. İlk program Main metoduyla başlar. 

using System;
class Program {
    static void Main() {
        Console.WriteLine("Merhaba Dünya!");
    }
}

  • Main: Programın giriş noktası.
  • Console.WriteLine: Ekrana çıktı yazar.

Soru: C#’ta programın giriş noktası nedir?



2. Değişkenler ve Veri Türleri

Değişkenler, veri saklar. Veri türleri: int, string, double. 

int yas = 25;
string isim = "Ahmet";
double boy = 1.75;

  • int: Tam sayı.
  • string: Metin.

Soru: Metin verisi için hangi veri türü kullanılır?



3. Operatörler

Operatörler, işlemler yapar: +, ==, &&. 

int a = 5 + 3;
bool b = a == 8;

  • +: Toplama.
  • ==: Eşitlik kontrolü.

Soru: Eşitlik kontrolü için hangi operatör kullanılır?



4. Koşullu İfadeler

if, else, switch ile koşullar kontrol edilir. 

int yas = 18;
if (yas >= 18) {
    Console.WriteLine("Yetişkin");
} else {
    Console.WriteLine("Çocuk");
}

  • if: Koşulu kontrol eder.
  • else: Alternatif durum.

Soru: Koşullu ifade için hangi anahtar kelime kullanılır?



5. Döngüler

for, while döngüleri tekrarlı işlemler yapar. 

for (int i = 0; i < 3; i++) {
    Console.WriteLine(i);
}

  • for: Belirli bir sayıda döner.
  • while: Koşul doğru olduğu sürece döner.

Soru: Belirli bir sayıda döngü için hangi anahtar kelime kullanılır?



6. Diziler

Diziler, birden fazla veriyi saklar. 

int[] sayilar = {1, 2, 3};
Console.WriteLine(sayilar[0]); // 1

  • []: Dizi elemanına erişir.
  • Sabit uzunluklu veri saklar.

Soru: Dizi elemanına erişmek için ne kullanılır?



7. String Sınıfı

string sınıfı, metin işlemleri için metotlar sunar. 

string metin = "Merhaba";
Console.WriteLine(metin.ToUpper()); // MERHABA

  • ToUpper: Büyük harfe çevirir.
  • Length: Uzunluğu döndürür.

Soru: Metni büyük harfe çeviren metot nedir?



8. Metotlar

Metotlar, tekrar kullanılabilir kod bloklarıdır. 

public int Topla(int a, int b) {
    return a + b;
}

  • return: Değer döndürür.
  • Parametreler isteğe bağlıdır.

Soru: Metottan değer döndürmek için hangi anahtar kelime kullanılır?



9. Sınıflar ve Nesneler

Sınıflar, nesnelerin şablonlarıdır. 

class Kisi {
    public string Isim;
    public Kisi(string isim) {
        Isim = isim;
    }
}
Kisi kisi = new Kisi("Ahmet");

  • class: Sınıf tanımlar.
  • new: Nesne oluşturur.

Soru: Nesne oluşturmak için hangi anahtar kelime kullanılır?



10. Kurucular

Kurucular, nesne oluşturulurken çağrılır. 

class Kisi {
    public string Isim;
    public Kisi(string isim) {
        Isim = isim;
    }
}

  • Kurucu, sınıf adıyla aynıdır.
  • this: Nesneye işaret eder.

Soru: Nesneye işaret eden anahtar kelime nedir?



11. Erişim Belirleyicileri

public, private erişimi kontrol eder. 

class Kisi {
    private string isim;
    public string GetIsim() {
        return isim;
    }
}

  • private: Sadece sınıf içinden erişilir.
  • public: Her yerden erişilir.

Soru: Sadece sınıf içinden erişilen belirleyici nedir?



12. Ad Alanları

Ad alanları, kodları organize eder. 

using System;
namespace Ornek {
    class Program {}
}

  • namespace: Ad alanı tanımlar.
  • using: Dış ad alanlarını çağırır.

Soru: Dış ad alanlarını çağırmak için hangi anahtar kelime kullanılır?



Orta Düzey Konular

Bu bölümde, C#’ın ileri düzey özelliklerini keşfedeceksiniz.

13. Kalıtım

Kalıtım, sınıfların özelliklerini miras almasını sağlar. 

class Hayvan {
    public virtual void Ses() {}
}
class Kopek : Hayvan {
    public override void Ses() {
        Console.WriteLine("Hav!");
    }
}

  • :: Kalıtımı sağlar.
  • virtual, override: Metot geçersiz kılma.

Soru: Kalıtım için hangi sembol kullanılır?



14. Polimorfizm

Polimorfizm, metotların farklı şekillerde çalışmasını sağlar. 

class Hayvan {
    public virtual void Ses() {}
}
class Kopek : Hayvan {
    public override void Ses() {
        Console.WriteLine("Hav!");
    }
}

  • Metot overriding: Aynı isim, farklı davranış.
  • Metot overloading: Aynı isim, farklı parametreler.

Soru: Aynı isimde farklı parametrelerle metot tanımlama nedir?



15. Arayüzler

Arayüzler, sınıfların uygulaması gereken metotları tanımlar. 

interface IHareket {
    void HareketEt();
}
class Araba : IHareket {
    public void HareketEt() {
        Console.WriteLine("Araba sürüyor.");
    }
}

  • interface: Metot şablonları tanımlar.
  • Sınıflar, arayüzleri uygular.

Soru: Metot şablonları tanımlamak için ne kullanılır?



16. Genel Tipler

Genel tipler, esnek ve yeniden kullanılabilir kod sağlar. 

class Depo {
    public T Veri;
}
Depo depo = new Depo();
depo.Veri = 42;

  • T: Genel tip parametresi.
  • Listeler, sözlükler gibi kullanılır.

Soru: Genel tip parametresi nasıl belirtilir?



17. LINQ

LINQ, veriler üzerinde sorgular yapmayı sağlar. 

using System.Linq;
int[] sayilar = {1, 2, 3, 4};
var ciftler = sayilar.Where(x => x % 2 == 0);

  • Where: Koşula göre filtreler.
  • System.Linq: LINQ için gerekli ad alanı.

Soru: LINQ için gerekli ad alanı nedir?
