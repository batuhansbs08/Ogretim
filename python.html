Başlangıç Seviyesi Konular

Bu bölümde, Python programlama dilinin temellerini öğreneceksiniz.

1. Python'a Giriş

Python, okunabilir ve çok yönlü bir programlama dilidir. İlk programınızı yazmak için print kullanılır. 

# Merhaba Dünya
print("Merhaba Dünya!")

  • print: Ekrana çıktı yazar.
  • #: Yorum satırı ekler.

Soru: Ekrana çıktı yazmak için hangi fonksiyon kullanılır?



2. Değişkenler ve Veri Türleri

Değişkenler, veri saklamak için kullanılır. Python’da veri türleri: int, str, float. 

isim = "Ahmet"  # str
yas = 25        # int
boy = 1.75      # float

  • =: Değer atar.
  • type(): Veri türünü döndürür.

Soru: Metin verisi için hangi veri türü kullanılır?



3. Operatörler

Operatörler, matematiksel ve mantıksal işlemler için kullanılır: +, ==, and. 

a = 5 + 3  # 8
b = a == 8  # True

  • +: Toplama veya string birleştirme.
  • ==: Eşitlik kontrolü.

Soru: Eşitlik kontrolü için hangi operatör kullanılır?



4. Koşullu İfadeler

if, else ile koşullu işlemler yapılır. 

yas = 18
if yas >= 18:
    print("Yetişkin")
else:
    print("Çocuk")

  • if: Koşulu kontrol eder.
  • else: Alternatif durum.

Soru: Koşullu ifade için hangi anahtar kelime kullanılır?



5. Döngüler

for ve while döngüleri tekrarlı işlemleri yapar. 

for i in range(3):
    print(i)  # 0, 1, 2

  • for: Belirli bir aralıkta döner.
  • while: Koşul doğru olduğu sürece döner.

Soru: Belirli bir aralıkta döngü için hangi anahtar kelime kullanılır?



6. Listeler

Listeler, birden fazla veriyi saklar. Değiştirilebilir (mutable). 

liste = [1, 2, 3]
liste.append(4)  # [1, 2, 3, 4]

  • append: Listeye eleman ekler.
  • [index]: Elemana erişir.

Soru: Listeye eleman eklemek için hangi metot kullanılır?



7. Tuple ve Set

Tuple değiştirilemez, set tekrar eden eleman içermez. 

tpl = (1, 2, 3)
st = {1, 2, 2}  # {1, 2}

  • tuple: Sabit veri saklar.
  • set: Benzersiz elemanlar saklar.

Soru: Benzersiz elemanlar için hangi veri yapısı kullanılır?



8. Sözlükler

Sözlükler, anahtar-değer çiftleri saklar. 

sozluk = {"isim": "Ahmet", "yas": 25}
print(sozluk["isim"])  # Ahmet

  • []: Anahtarla değere erişir.
  • dict: Sözlük oluşturur.

Soru: Sözlükte değere erişmek için ne kullanılır?



9. Fonksiyonlar

Fonksiyonlar, tekrar kullanılabilir kod bloklarıdır. 

def topla(a, b):
    return a + b
print(topla(3, 4))  # 7

  • def: Fonksiyon tanımlar.
  • return: Değer döndürür.

Soru: Fonksiyon tanımlamak için hangi anahtar kelime kullanılır?



10. String Metotları

String’ler üzerinde işlem yapan metotlari: upper, split. 

metin = "Merhaba Dünya"
print(metin.upper())  # MERHABA DÜNYA

  • upper: Büyük harfe çevirir.
  • split: String’i böler.

Soru: String’i büyük harfe çeviren metot nedir?



11. Hata Yönetimi

try-except ile hatalar yakalanır. 

try:
    sayi = int("abc")
except ValueError:
    print("Geçersiz sayı")

  • try: Hata olabilecek kodu çalıştırır.
  • except: Hata yakalar.

Soru: Hataları yakalamak için hangi yapı kullanılır?



12. Dosya İşlemleri

Dosyaları okumak ve yazmak için open kullanılır. 

with open("dosya.txt", "w") as dosya:
    dosya.write("Merhaba")

  • open: Dosya açar.
  • "w": Yazma modu.

Soru: Dosya yazma modu için hangi parametre kullanılır?



Orta Düzey Konular

Bu bölümde, Python’un ileri düzey özelliklerini keşfedeceksiniz.

13. Modüller ve Paketler

Modüller, kodu organize eder. import ile kullanılır. 

import math
print(math.sqrt(16))  # 4.0

  • import: Modül çağırır.
  • math: Matematiksel işlemler sağlar.

Soru: Modül çağırmak için hangi anahtar kelime kullanılır?



14. Nesne Yönelimli Programlama

Sınıflar ve nesneler, veri ve işlevselliği birleştirir. 

class Kisi:
    def __init__(self, isim):
        self.isim = isim
kisi = Kisi("Ahmet")
print(kisi.isim)  # Ahmet

  • class: Sınıf tanımlar.
  • __init__: Kurucu metot.

Soru: Sınıf tanımlamak için hangi anahtar kelime kullanılır?



15. Lambda Fonksiyonları

Lambda, tek satırlık anonim fonksiyonlar oluşturur. 

topla = lambda x, y: x + y
print(topla(3, 4))  # 7

  • lambda: Anonim fonksiyon tanımlar.
  • Kısa ve geçici fonksiyonlar için kullanılır.

Soru: Anonim fonksiyon tanımlamak için hangi anahtar kelime kullanılır?



16. List Comprehension

List comprehension, kısa yoldan liste oluşturur. 

kareler = [x**2 for x in range(5)]
print(kareler)  # [0, 1, 4, 9, 16]

  • Kompakt döngü yerine geçer.
  • Hızlı ve okunabilir.

Soru: List comprehension hangi amaçla kullanılır?



17. Veritabanı İşlemleri

sqlite3 ile veritabanı işlemleri yapılır. 

import sqlite3
conn = sqlite3.connect("ornek.db")
conn.close()

  • sqlite3: Hafif veritabanı sağlar.
  • connect: Veritabanına bağlanır.

Soru: SQLite veritabanına bağlanmak için hangi modül kullanılır?
