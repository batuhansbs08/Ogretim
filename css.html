Başlangıç Seviyesi Konular

Bu bölümde, CSS ile web sayfalarını biçimlendirmenin temellerini öğreneceksiniz.

1. CSS'ye Giriş

CSS, web sayfalarının görünümünü kontrol eder. Temel söz dizimi seçici ve özelliklerden oluşur. 

p {
    color: blue;
}

  • p: Seçici, paragraf etiketini hedefler.
  • color: Metin rengini ayarlar.

Soru: CSS’de metin rengini ayarlayan özellik nedir?



2. Seçiciler

Seçiciler, hangi HTML elemanlarının biçimlendirileceğini belirler. 

.sinif {
    font-weight: bold;
}
#id {
    color: red;
}

  • .sinif: Sınıf seçicisi.
  • #id: ID seçicisi.

Soru: Sınıf seçicisi nasıl başlar?



3. Renk ve Arka Plan

color ve background özellikleri görünümü değiştirir. 

div {
    background-color: #f0f0f0;
    color: #333;
}

  • background-color: Arka plan rengi.
  • color: Metin rengi.

Soru: Arka plan rengini ayarlayan özellik nedir?



4. Yazı Tipi ve Metin

Yazı tipi ve metin özellikleri: font-family, text-align. 

p {
    font-family: Arial, sans-serif;
    text-align: center;
}

  • font-family: Yazı tipini belirler.
  • text-align: Metin hizalamasını ayarlar.

Soru: Metin hizalamasını ayarlayan özellik nedir?



5. Kenar Çizgileri

border özelliği, elemanlara çerçeve ekler. 

div {
    border: 1px solid black;
}

  • border: Kalınlık, stil ve renk tanımlar.
  • Örnek: solid, dashed.

Soru: Çerçeve eklemek için hangi özellik kullanılır?



6. Margin ve Padding

margin dış boşluk, padding iç boşluk sağlar. 

div {
    margin: 10px;
    padding: 15px;
}

  • margin: Elemanlar arası boşluk.
  • padding: İçerik ile kenar arası boşluk.

Soru: İç boşluk için hangi özellik kullanılır?



7. Kutu Modeli

Kutu modeli, elemanların boyutlarını tanımlar. 

div {
    width: 200px;
    height: 100px;
    box-sizing: border-box;
}

  • box-sizing: Boyut hesaplama yöntemini belirler.
  • İçerik, padding, border içerir.

Soru: Boyut hesaplama yöntemini belirleyen özellik nedir?



8. Display Özelliği

display, elemanların yerleşim türünü belirler. 

span {
    display: block;
}

  • block: Yeni satırda başlar.
  • inline: Aynı satırda kalır.

Soru: Yeni satırda başlayan yerleşim türü nedir?



9. Pozisyonlandırma

position, elemanların yerleşimini kontrol eder. 

div {
    position: absolute;
    top: 10px;
}

  • absolute: Ebeveyne göre konumlanır.
  • relative: Normal konumuna göre ayarlanır.

Soru: Ebeveyne göre konumlandırma için hangi değer kullanılır?



10. Float ve Clear

float, elemanları sola veya sağa kaydırır. 

img {
    float: left;
}
div {
    clear: both;
}

  • float: Elemanı kaydırır.
  • clear: Kayan elemanları temizler.

Soru: Kayan elemanları temizlemek için hangi özellik kullanılır?



11. Bağlantı Stilleri

Bağlantılar, a etiketi ile biçimlendirilir. 

a:hover {
    color: blue;
    text-decoration: underline;
}

  • :hover: Fare üzerine geldiğinde stil uygular.
  • text-decoration: Alt çizgi gibi stiller ekler.

Soru: Fare üzerine geldiğinde stil uygulayan durum nedir?



12. Liste Stilleri

list-style, liste işaretlerini biçimlendirir. 

ul {
    list-style-type: square;
}

  • list-style-type: İşaret türünü belirler.
  • Örnek: disc, square.

Soru: Liste işaretini belirleyen özellik nedir?



Orta Düzey Konular

Bu bölümde, CSS’nin modern ve gelişmiş özelliklerini keşfedeceksiniz.

13. Flexbox

Flexbox, esnek düzenler oluşturur. 

.container {
    display: flex;
    justify-content: center;
}

  • display: flex: Esnek kutu düzeni.
  • justify-content: Yatay hizalamayı ayarlar.

Soru: Flexbox düzeni için hangi özellik kullanılır?



14. CSS Grid

Grid, iki boyutlu düzenler oluşturur. 

.grid {
    display: grid;
    grid-template-columns: 1fr 1fr;
}

  • display: grid: Izgara düzeni.
  • grid-template-columns: Sütun boyutlarını tanımlar.

Soru: Izgara düzeni için hangi özellik kullanılır?



15. Geçişler

transition, stil değişikliklerini yumuşatır. 

button {
    transition: background-color 0.3s;
}
button:hover {
    background-color: blue;
}

  • transition: Özellik, süre ve zamanlama tanımlar.
  • Hareketli efektler için kullanılır.

Soru: Stil değişikliklerini yumuşatan özellik nedir?



16. Dönüşümler

transform, elemanları döndürür, ölçekler veya kaydırır. 

div {
    transform: rotate(45deg);
}

  • rotate: Derece cinsinden döndürür.
  • scale: Boyutları değiştirir.

Soru: Elemanları döndüren dönüşüm fonksiyonu nedir?



17. CSS Animasyonları

animation, özel animasyonlar oluşturur. 

@keyframes move {
    from { transform: translateX(0); }
    to { transform: translateX(100px); }
}
div {
    animation: move 2s infinite;
}

  • @keyframes: Animasyon adımlarını tanımlar.
  • animation: Süre, tekrar gibi ayarları yapar.

Soru: Animasyon adımlarını tanımlayan yapı nedir?
