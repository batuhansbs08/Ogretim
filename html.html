Başlangıç Seviyesi Konular

Bu bölümde, HTML ile web geliştirme yolculuğunuza başlayacak ve temel yapıları öğreneceksiniz.

1. HTML Temel Yapı

HTML, web sayfalarının iskeletini oluşturur. Temel yapı <html>, <head> ve <body> etiketlerinden oluşur. 

<!DOCTYPE html>
<html lang="tr">
<head>
  <meta charset="UTF-8">
  <title>Başlık</title>
</head>
<body>
  <h1>Merhaba Dünya!</h1>
</body>
</html>

  • <!DOCTYPE html>: HTML5 belgesi olduğunu belirtir.
  • <html>: Tüm içeriği sarar.

Soru: HTML5 belgesini tanımlayan ifade nedir?



2. Etiketler

Etiketler, HTML'nin yapı taşlarıdır. Açılış ve kapanış etiketleri içeriği tanımlar. 

<p>Bu bir paragraf.</p>
<h1>Başlık</h1>

  • <p>: Paragraf oluşturur.
  • <h1>: En büyük başlık etiketi.

Soru: Paragraf oluşturmak için hangi etiket kullanılır?



3. Öznitelikler

Öznitelikler, etiketlere ek bilgi sağlar. Örneğin, id, class. 

<div id="ana" class="stil">İçerik</div>

  • id: Benzersiz kimlik tanımlar.
  • class: Stil veya gruplandırma için kullanılır.

Soru: Benzersiz kimlik tanımlamak için hangi öznitelik kullanılır?



4. Bağlantılar

<a> etiketi, bağlantılar oluşturur. href özniteliği hedefi belirtir. 

<a href="https://ornek.com">Tıkla</a>

  • href: Bağlantının hedef adresi.
  • target="_blank": Yeni sekmede açar.

Soru: Bağlantı oluşturmak için hangi etiket kullanılır?



5. Resimler

<img> etiketi, resimleri ekler. src ve alt öznitelikleri kullanılır. 

<img src="resim.jpg" alt="Açıklama">

  • src: Resmin dosya yolu.
  • alt: Resim yüklenmezse gösterilen metin.

Soru: Resim eklemek için hangi etiket kullanılır?



6. Listeler

HTML'de sıralı (<ol>) ve sırasız (<ul>) listeler kullanılır. 

<ul>
  <li>Madde 1</li>
  <li>Madde 2</li>
</ul>

  • <ul>: Sırasız liste (noktalı).
  • <ol>: Sıralı liste (numaralı).

Soru: Sırasız liste oluşturmak için hangi etiket kullanılır?



7. Tablolar

<table> etiketi, veri tabloları oluşturur. <tr>, <td>, <th> kullanılır. 

<table>
  <tr>
    <th>Başlık</th>
    <td>Veri</td>
  </tr>
</table>

  • <tr>: Satır tanımlar.
  • <th>: Başlık hücresi.

Soru: Tablo satırı tanımlamak için hangi etiket kullanılır?



8. Formlar

<form> etiketi, kullanıcı girişi toplar. <input> ve <label> kullanılır. 

<form>
  <label for="isim">İsim:</label>
  <input type="text" id="isim">
</form>

  • <input>: Kullanıcı girişi alır.
  • <label>: Giriş alanını tanımlar.

Soru: Kullanıcı girişi almak için hangi etiket kullanılır?

9. Div ve Span

<div> ve <span>, içeriği gruplandırmak için kullanılır. 

<div>Blok içerik</div>
<span>Satır içi içerik</span>

  • <div>: Blok seviyesinde gruplama.
  • <span>: Satır içi gruplama.

Soru: Satır içi gruplama için hangi etiket kullanılır?



10. Semantik Etiketler

Semantik etiketler, içeriğin anlamını belirtir. Örneğin, <header>, <footer>. 

<header>Site Başlığı</header>
<main>Ana İçerik</main>

  • <header>: Üst bilgi alanı.
  • <main>: Ana içerik alanı.

Soru: Ana içerik alanı için hangi etiket kullanılır?



11. Meta Etiketler

<meta> etiketleri, sayfa hakkında bilgi sağlar. Örneğin, karakter seti, açıklama. 

<meta charset="UTF-8">
<meta name="description" content="Web sitesi">

  • charset: Karakter kodlamasını tanımlar.
  • name="description": Arama motorları için açıklama.

Soru: Karakter kodlamasını tanımlayan öznitelik nedir?



12. Temel Olaylar

HTML'de olaylar, kullanıcı etkileşimlerini yakalar. Örneğin, onclick. 

<button onclick="alert('Tıklandı!')">Tıkla</button>

  • onclick: Tıklama olayını yakalar.
  • JavaScript ile dinamik davranış eklenir.

Soru: Tıklama olayını yakalamak için hangi öznitelik kullanılır?



Orta Düzey Konular

Bu bölümde, HTML'nin daha gelişmiş özelliklerini ve HTML5'in sunduğu modern imkanları öğreneceksiniz.

13. HTML5 Canvas

<canvas> etiketi, JavaScript ile grafik çizimi için kullanılır. Oyunlar veya veri görselleştirme için idealdir. 

<canvas id="myCanvas" width="200" height="100"></canvas>
<script>
  var canvas = document.getElementById("myCanvas");
  var ctx = canvas.getContext("2d");
  ctx.fillStyle = "blue";
  ctx.fillRect(50, 25, 100, 50);
</script>

  • <canvas>: Çizim alanı oluşturur.
  • getContext("2d"): 2D çizim bağlamını sağlar.

Soru: Canvas’ta 2D çizim için hangi yöntem kullanılır?



14. Veri Öznitelikleri

data-* öznitelikleri, özel veri saklamak için kullanılır. JavaScript ile erişilebilir. 

<div data-rol="admin">Kullanıcı</div>
<script>
  var element = document.querySelector('[data-rol]');
  console.log(element.dataset.rol); // admin
</script>

  • data-*: Özel veri özniteliği tanımlar.
  • dataset: JavaScript’te veriye erişir.

Soru: Veri özniteliği nasıl başlar?



15. Mikro Veri

Mikro veri, arama motorlarının içeriği anlamasını sağlar. itemscope ve itemprop kullanılır. 

<div itemscope itemtype="http://schema.org/Person">
  <span itemprop="name">Ahmet Yılmaz</span>
  <span itemprop="jobTitle">Geliştirici</span>
</div>

  • itemscope: Mikro veri bloğunu tanımlar.
  • itemprop: Özelliği belirtir.

Soru: Mikro veri bloğunu tanımlayan öznitelik nedir?



16. Web Storage

Web Storage, tarayıcıda veri saklamak için kullanılır (localStorage, sessionStorage). 

<button onclick="saveData()">Kaydet</button>
<script>
  function saveData() {
    localStorage.setItem("kullanici", "Ahmet");
    alert("Kaydedildi: " + localStorage.getItem("kullanici"));
  }
</script>

  • localStorage: Kalıcı veri saklar.
  • sessionStorage: Sekme kapandığında silinir.

Soru: Kalıcı veri saklamak için hangi nesne kullanılır?



17. Erişilebilirlik (ARIA)

ARIA (Accessible Rich Internet Applications), engelli kullanıcılar için erişilebilirlik sağlar. 

<button aria-label="Kapat">X</button>
<div role="alert">Hata mesajı</div>

  • aria-label: Ekran okuyucular için açıklama.
  • role: Elemanın işlevini tanımlar.

Soru: Ekran okuyucular için açıklama sağlayan öznitelik nedir?
