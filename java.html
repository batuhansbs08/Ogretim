Başlangıç Seviyesi Konular

Bu bölümde, Java programlama dilinin temellerini öğreneceksiniz.

1. Java'ya Giriş

Java, platform bağımsız bir dildir. İlk program main metoduyla başlar. 

public class Merhaba {
    public static void main(String[] args) {
        System.out.println("Merhaba Dünya!");
    }
}

  • main: Programın giriş noktası.
  • System.out.println: Ekrana çıktı yazar.

Soru: Java’da programın giriş noktası nedir?



2. Değişkenler ve Veri Türleri

Değişkenler, veri saklar. Veri türleri: int, String, double. 

int yas = 25;
String isim = "Ahmet";
double boy = 1.75;

  • int: Tam sayı.
  • String: Metin.

Soru: Metin verisi için hangi veri türü kullanılır?



3. Operatörler

Operatörler, işlemler yapar: +, ==, &&. 

int a = 5 + 3;
boolean b = a == 8;

  • +: Toplama.
  • ==: Eşitlik kontrolü.

Soru: Eşitlik kontrolü için hangi operatör kullanılır?



4. Koşullu İfadeler

if, else, switch ile koşullar kontrol edilir. 

int yas = 18;
if (yas >= 18) {
    System.out.println("Yetişkin");
} else {
    System.out.println("Çocuk");
}

  • if: Koşulu kontrol eder.
  • else: Alternatif durum.

Soru: Koşullu ifade için hangi anahtar kelime kullanılır?



5. Döngüler

for, while döngüleri tekrarlı işlemler yapar. 

for (int i = 0; i < 3; i++) {
    System.out.println(i);
}

  • for: Belirli bir sayıda döner.
  • while: Koşul doğru olduğu sürece döner.

Soru: Belirli bir sayıda döngü için hangi anahtar kelime kullanılır?



6. Diziler

Diziler, birden fazla veriyi saklar. 

int[] sayilar = {1, 2, 3};
System.out.println(sayilar[0]); // 1

  • []: Dizi elemanına erişir.
  • Sabit uzunluklu veri saklar.

Soru: Dizi elemanına erişmek için ne kullanılır?



7. String Sınıfı

String sınıfı, metin işlemleri için metotlar sunar. 

String metin = "Merhaba";
System.out.println(metin.toUpperCase()); // MERHABA

  • toUpperCase: Büyük harfe çevirir.
  • length: Uzunluğu döndürür.

Soru: Metni büyük harfe çeviren metot nedir?



8. Metotlar

Metotlar, tekrar kullanılabilir kod bloklarıdır. 

public int topla(int a, int b) {
    return a + b;
}

  • return: Değer döndürür.
  • Parametreler isteğe bağlıdır.

Soru: Metottan değer döndürmek için hangi anahtar kelime kullanılır?



9. Sınıflar ve Nesneler

Sınıflar, nesnelerin şablonlarıdır. 

class Kisi {
    String isim;
    Kisi(String isim) {
        this.isim = isim;
    }
}
Kisi kisi = new Kisi("Ahmet");

  • class: Sınıf tanımlar.
  • new: Nesne oluşturur.

Soru: Nesne oluşturmak için hangi anahtar kelime kullanılır?



10. Kurucular

Kurucular, nesne oluşturulurken çağrılır. 

class Kisi {
    String isim;
    Kisi(String isim) {
        this.isim = isim;
    }
}

  • Kurucu, sınıf adıyla aynıdır.
  • this: Nesneye işaret eder.

Soru: Nesneye işaret eden anahtar kelime nedir?



11. Erişim Belirleyicileri

public, private erişimi kontrol eder. 

class Kisi {
    private String isim;
    public String getIsim() {
        return isim;
    }
}

  • private: Sadece sınıf içinden erişilir.
  • public: Her yerden erişilir.

Soru: Sadece sınıf içinden erişilen belirleyici nedir?



12. Paketler

Paketler, kodları organize eder. 

package ornek;
import java.util.Scanner;

  • package: Paket tanımlar.
  • import: Dış sınıfları çağırır.

Soru: Dış sınıfları çağırmak için hangi anahtar kelime kullanılır?



Orta Düzey Konular

Bu bölümde, Java’nın ileri düzey özelliklerini keşfedeceksiniz.

13. Kalıtım

Kalıtım, sınıfların özelliklerini miras almasını sağlar. 

class Hayvan {
    void ses() {}
}
class Kopek extends Hayvan {
    void ses() {
        System.out.println("Hav!");
    }
}

  • extends: Kalıtımı sağlar.
  • Üst sınıf özellikleri miras alınır.

Soru: Kalıtım için hangi anahtar kelime kullanılır?



14. Polimorfizm

Polimorfizm, metotların farklı şekillerde çalışmasını sağlar. 

class Hayvan {
    void ses() {}
}
class Kopek extends Hayvan {
    void ses() {
        System.out.println("Hav!");
    }
}

  • Metot overriding: Aynı isim, farklı davranış.
  • Metot overloading: Aynı isim, farklı parametreler.

Soru: Aynı isimde farklı parametrelerle metot tanımlama nedir?



15. Soyut Sınıflar ve Arayüzler

Soyut sınıflar ve arayüzler, şablonlar oluşturur. 

abstract class Hayvan {
    abstract void ses();
}
interface Hareket {
    void hareketEt();
}

  • abstract: Somut olmayan metotlar tanımlar.
  • interface: Sözleşme gibidir.

Soru: Sözleşme tanımlamak için hangi yapı kullanılır?



16. İstisna Yönetimi

try-catch ile hatalar yakalanır. 

try {
    int sayi = Integer.parseInt("abc");
} catch (NumberFormatException e) {
    System.out.println("Geçersiz sayı");
}

  • try: Hata olabilecek kodu çalıştırır.
  • catch: Hata yakalar.

Soru: Hataları yakalamak için hangi yapı kullanılır?



17. Koleksiyonlar

Koleksiyonlar, dinamik veri saklar: ArrayList, HashMap. 

import java.util.ArrayList;
ArrayList liste = new ArrayList<>();
liste.add("Ahmet");

  • ArrayList: Dinamik liste.
  • HashMap: Anahtar-değer çiftleri.

Soru: Dinamik liste için hangi sınıf kullanılır?
